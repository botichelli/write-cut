# Write-Cut ✂️

Інструмент для редагування текстів за методологією інформаційного стилю з книги **«Пиши, скорочуй»** (Максим Ільяхов і Людмила Саричева).

Прибирає воду, канцеляризми та порожні оцінки. Залишає суть, факти і структуру.

---

## Що всередині

```
write-cut/
├── PROMPT.md          # Системний промпт — для ChatGPT, Gemini та будь-якого LLM
├── write-cut.skill    # Skill-файл для Claude (Cowork / Claude Code)
└── skill/
    ├── SKILL.md       # Вихідний код skill'а для Claude
    └── references/
        └── examples.md  # Розгорнуті приклади до/після
```

---

## Як використовувати

### ChatGPT / Gemini / будь-який LLM
1. Відкрий `PROMPT.md`
2. Скопіюй весь вміст
3. Встав у поле «System instructions» (або як перше повідомлення в чаті)
4. Надішли текст, який потрібно відредагувати

### Claude (Cowork)
1. Завантаж `write-cut.skill`
2. У Cowork натисни **«Copy to your skills»**
3. Після цього skill спрацьовує автоматично — просто напиши «відредагуй цей текст» або «зроби краще»

### Claude Code
```bash
# Скопіюй папку skill/ до директорії своїх skills
cp -r skill/ ~/.claude/skills/write-cut/
```

## Що вміє

Редагування в три рівні:

**Рівень 1 — Слова.** Видаляє 8 категорій сміття: вступні паразити, оцінки без доказів, підсилювачі, канцеляризми, штампи, евфемізми, відіменникові дієслова, невизначеність.

**Рівень 2 — Структура.** Активний стан замість пасивного, одна думка на речення, абзаци з чіткою архітектурою, функціональні заголовки, конкретні факти.

**Рівень 3 — Зміст.** Для рекламних і презентаційних текстів: характеристики у переваги, докази замість тверджень, фокус на користь для читача.
